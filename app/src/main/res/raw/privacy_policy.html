Access data

Escalero uses Google Firebase technology in online mode (Google Inc., 1600 Amphitheater Parkway, Montain View, CA 94043, USA, "Google"). Firebase is part of Google Cloud Platform. The following services are used:

Authentication: Sign in a player through a Google Account

Realtime Database: player, match and notification data are stored for the technical process of the game.

Cloud Messaging: for player notifications (invitation, game over, game resume)

Google Firebase uses servers with an EU location for these services wherever possible. However, it can not be ruled out that data will also be transmitted to the USA. Google has joined the EU-US Privacy Shield, an EU-US data protection agreement. More information about Google Firebase and privacy can be found here:

https://firebase.google.com/support/privacy/ 

Personal data

Personal data is information that can be used to determine a person, i.e. information that can be traced back to a person.

Escalero requires the following critical data and is stored in the Firebase Database:

User ID (Firebase.uid): a key required to uniquely identify a player.

ID Token: Identity credential token between the Escalero app and Firebase app. Required for player notifications (cloud messaging).

Hardware ID: to prevent multiple logins of a player is a program-internal check. For security reasons, only the first 7 digits of the hardware ID are stored.

No further personal data will be stored in the App Escalero. Players will not have access to the critical data listed above. The communication of the players takes place exclusively over the player name. This is recorded when the program is started for the first time and can be changed at any time.

Installation of the app

The Escalero app is available through Google Play Store. The subscription requires prior registration. I have no control over the collection, processing and use of personal data in connection with the App Store. The responsible body is the operator Google Play Store.
