Information

Eskalero ist ein Würfelspiel der Kategorie "Strategie und Zufall" und wird mit einem Satz von fünf Würfeln gespielt. 

Das Spiel kann Online oder Offline gespielt werden. Eine Beschreibung des Online-Spieles erfolgt am Ende dieser Spielanleitung.   

Das Spiel zu zweit

Allgemeines

Die Spieler notieren ihre Resultate in einer Tabelle mit zehn Reihen (Zeilen) und sechs Kolonnen (Spalten).

Die ersten sechs Zeilen der (eigentlichen) Tabelle sind den sogenannten Bildern, die weiteren vier den Kombinationen gewidmet. In einer Abschlusszeile werden die Spaltensummen/Differenzen angezeigt. Jedem Spieler (gekennzeichnet durch die Farbe blau oder gelb) stehen so für das Eintragen seiner Resultate drei Spalten zur Verfügung.

In jeder Runde darf drei mal gewürfelt werden:

der Wurfvorgang wird gestartet durch das Berühren des Becher-Symbols. Durch das Berühren der Würfel werden diese gehalten oder für den nächsten Wurfvorgang freigegeben.

Ein Spieler hat die Möglichkeit, nach dem ersten Wurf oder auch nach dem zweiten Wurf die Würfel beliebig zu halten oder freizugeben. Nach dem letzten Wurf muss der Spieler entscheiden, was er in der Tabelle einträgt.

Die Kombinationen (S, F, P, G):

S: Straße (A-K-D-B-10 oder K-D-B-10-9). Wert 20 Punkte, serviert 25 Punkte

F: Full House (drei Gleiche und ein Paar). Wert 30 Punkte, serviert 35 Punkte

P: Poker (vier Gleiche, four of a kind). Wert 40 Punkte, serviert 45 Punkte.

G: Grande (fünf Gleiche, five of a kind). Wert 50 Punkte, serviert 80 Punkte.

Eine Servierung (Wurf aus der Hand) erhöht den Grundwert von Straße, Full House und Poker um jeweils fünf Punkte, im Falle eines Grande um 30 Punkte(die Bonuspunkte für eine Servierung können in den Einstellungen geänderte werden).

Bei einer Servierung ist es gleichgültig, ob es sich um den ersten, zweiten oder dritten Wurf eines Spielers handelt, wesentlich für einen Wurf aus der Hand ist, dass alle fünf Würfel zugleich geworfen werden.

Die Bilder (9, 10, B, D, K, A):

Abgesehen von den obigen Kombinationen muss man als Spieler versuchen, die verschiedenen Bilder von Neun bis Ass möglichst oft zu erreichen, das Optimum ist fünfmal die Neun, fünfmal die Zehn usw. Hierfür erhält man wie folgt Punkte: Man multipliziert die Anzahl der Würfel, die das Bild zeigen, mit der Punktezahl für das betreffende Bild.

Die Punktewerte der einzelnen Bilder sind: Eine Neun zählt einen Punkt, eine Zehn zählt zwei Punkte, Bube drei, Dame vier, König fünf und das Ass sechs Punkte; so zählen etwa vier Buben zwölf Punkte oder drei Könige fünfzehn Punkte.

Das Notieren der Punkte:

Jeder Spieler trägt am Ende einer Runde die erzielten Punkte in der Tabelle in der dem Bild bzw. der Kombination entsprechenden Reihe (Zeile) und je nach Wunsch in der ersten, zweiten oder dritten Kolonne (Spalte) ein. In jedes Feld der Tabelle darf nur einmal eingetragen werden, einmal eingetragene Punkte dürfen nicht mehr geändert oder etwa in eine andere Kolonne verschoben werden.

Bevor der nächste Spieler würfelt, kann der letzte Eintrag, durch Auswahl in der Tabelle(rot markiert), gelöscht werden.

Hat ein Spieler in einer Runde besonders schlecht gewürfelt und kann keine gewinnbringende Eintragung vornehmen, so darf er, um den Verlust klein zu halten, ein Feld der Tabelle streichen, d.h. in das entsprechende Feld eine Null eintragen, z.B. in das Feld für die Neuner in der ersten Kolonne.

Die Abrechnung:

Der Sieger einer Kolonne ist derjenige Spieler, der die höhere Gesamtpunktezahl erzielen konnte.

Für den Gewinn der einzelnen Kolonnen werden wie folgt Punkte vergeben:

erste Kolonne: 1 Punkt

zweite Kolonne: 2 Punkte

dritte Kolonne: 4 Punkte

Endet eine Kolonne unentschieden, so schreibt keiner der beiden Spieler dafür einen Punkt.

Gewinnt ein Spieler alle drei Kolonnen, so erhält er noch einen zusätzlichen Bonus von 3 Punkten.

Die Punkte für die Kolonnen sowie die Bonus Punkte können in den Einstellungen geändert werden.

Die Abrechnung wird nach Beendigung des Spieles (alle Einträge in der Tabelle sind erfolgt) angezeigt.

Das Spiel zu dritt

Nehmen drei Personen am Spiel teil, so spielt jeder für sich gegen die beiden anderen. Der Sieger einer Kolonne erhält von jedem der beiden Gegner die Punkte so wie beim Spiel zu zweit ausbezahlt.

Das Spiel zu viert oder mit mehr Personen

Nehmen vier oder mehr Personen an einer Partie teil, so spricht man von einem Doppel. Es werden Teams von jeweils 2 Spielern gebildet. Ist die Teilnehmerzahl ungerade, so spielt ein Spieler für zwei Personen. Maximal können 6 Personen an einem Spiel teilnehmen (Doppel | 3 Teams).

Jedes Team erhält drei Kolonnen, jeder der beiden Partner darf je Runde bis zu dreimal Würfeln, es darf aber nur eines der beiden Ergebnisse je Runde in der Tabelle eingetragen werden. Das Wurfergebnis des ersten Doppelspielers kann im Würfelbrett mit den Tasten rechts oben, im 2D-Modus mit der Taste D1, gehalten werden. In jeder Kolonne gibt es ein Siegerpaar (Team).

Vor Spielbeginn wird die Abrechnungsform (pro Team oder je Spielpartner) festgelegt.

Resultat-Eingabe

Das Programm kann auch nur zum Eintragen der Werte verwendet werden (der Wurfvorgang erfolgt manuell). In den Einstellungen wird die Option "Resultat-Eingabe" gesetzt.

Eingabe eines Wurfergebnisses:

Auswahl eines Eintrages in der Tabelle (Bild oder Kombination)eine Auswahl-Taste betätigen

Sondertasten in der Tabelle

Von den vier Ecken der Tabelle (rosa) können Spezialfunktionen aufgerufen werden:

oben links: Anzeige der Tabelleneinträge nach Spalten oder Spieler. Hier wird auch die Anzahl der noch ausstehenden Würfe angezeigt

oben rechts: Bildschirm-Drehung aktivieren/deaktivieren

unten links: Anzeige der Spalten-Ergebnisse nach Summe oder Differenz

unten rechts: Offline: die aktuelle Abrechnung wird angezeigt, Online: Spiel beginnen / Aktuelles Spiel

Speichern und laden von Spielen

Speichern: das aktuelle Spiel kann auf der SD-Karte gespeichert werden, Datei-Endung: .pgnd. Pro Datei können bis zu 50 Spiele gespeichert werden. Der Aufruf der Funktion erfolgt über die Menü-Taste. Datei-Ordner und -Name können geändert werden. Der Vorgang wird mit der OK-Taste abgeschlossen.

Laden: Der Aufruf der Funktion erfolgt über die Menü-Taste. Datei-Ordner und -Name können ausgewählt werden. Aus der angezeigten Liste wird ein Spiel gewählt und mit der OK-Taste der Vorgang abgeschlossen.

Online spielen

Eskalero verwendet die Entwicklungs-Plattform Firebase für die Daten-Kommunikation im Internet.

Derzeit wird das Eskalero-Spiel nur mit den Optionen "Einzel" und "zwei Spieler" gespielt. Die Spiel-Währung ist: EP (Eskalero-Punkte), die Kolonnen-Bewertung: 1-2-4, Bonus: 3 EP.

Nachstehend sind die Vorausetzungen für das Online-Spiel, die wichtigsten Funktionen, sowie weitere Hinweise, näher beschrieben.

Voraussetzungen

  • gute Internetverbindung
  • die Eskalero-App im Online-Modus
  • Google Benutzerkonto
  • beide Spieler müssen angemeldet sein

Funktionen:

Spiel beginnen: Ein Dialog, mit folgenden Funktionen, wird angezeigt:

Anmelden:  mit dem Google Benutzerkonto erfolgt eine Registrierung auf dem Firebase Server. Bei erfolgreicher Anmeldung wird der Spielername angezeigt und die Funktion Abmelden angezeigt. Der Spielername kann geändert werden.

Spieler-Suche: Es wird eine Spieler-Liste nach den Kategorien: "Online", "Oft aktive" und "Bestenliste" angezeigt. Bei Auswahl eines Spielers wird ein Einladungs-Dialog angezeigt und mit der Auswahl der Taste "Einladung" der Einladungs-Vorgang gestartet. Es erfolgt eine Benachrichtigung des zuvor ausgewählten Spielers. Nimmt der Spieler die Einladung an, kann das Spiel beginnen.

Schnelles Spiel: Wenn ein Spieler die "Schnelles Spiel" Taste drückt werden alle Spieler die Online sind davon informiert (Einladungs-Dialog). Nimmt ein Spieler die Einladung an, kann das Spiel nach erfolgter Verbindung beginnen.

Spiele-Verwaltung: Es werden alle offenen Spiele eines Spielers in einer Liste angezeigt. Ein Spiel wird erst gespeichert wenn zumindest ein Eintrag in der Tabelle erfolgt ist. Es werden maximal zehn Spiele verwaltet. Wird ein elftes Spiel begonnen, wird automatisch das älteste Spiel gelöscht. Nachfolgend eine genauere Beschreibung der Funktion Spiele-Verwaltung:

Anzeige (pro Listen-Eintrag):

Kontrollkästchen (Auswahl eines Spieles)

Spielernamen

Spiel-Status

Zeit (letzte Aktion)     

Aktionen

Zurück: Dialog wird beendet

Löschen: die oben ausgewählten Spiele werden gelöscht, der Dialog wird beendet

Fortsetzen: das ausgewählte Spiel wird geladen, der Dialog beendet, der Gegner erhält eine Benachrichtigung und bei erfolgter Verbindung kann das Spiel fortgesetzt werden 

Aktuelles Spiel:  der Vorgang kann vom Menü oder von der Firebase-Taste (Tabelle rechts unten) gestartet werden.

Wenn erforderlich, erfolgt eine Datensynchronisierung mit dem Firebase-Server. Es werden die aktuellen Spieldaten angezeigt. Folgende Aktionen sind möglich:

Abrechnung : der aktueller Spielstand wird angezeigt

Löschen: das Spiel wird gelöscht (keine weiteren Aktionen mehr möglich!)

Pause: das aktuelle Spiel wird beendet und, wenn möglich, der Gegner verständigt. Das Spiel kann später mit der Funktion "Spiele-Verwaltung" fortgesetzt werden.

Spiele-Verwaltung: Beschreibung siehe oben

Weitere Hinweise

  • nach einem Eintrag in der Tabelle muss dieser noch mit der OK-Taste (rechts unten) bestätigt werden!
  • eine Synchronisierung der Daten beider Spieler sollte innerhalb kurzer Zeit erfolgen. Mit der Funktion "Aktuelles Spiel" erfolgt, wenn erforderlich, eine manuelle Aktualisierung der Daten.
  • sollte das Spiel jedoch einmal hängen (kann z.B. bei schlechter Internet-Verbindung passieren) kann man das Spiel später mit der Funktion "Spiele-Verwaltung" fortsetzen.

Achtung: ist ein Spiel auch nur von einen Spieler gelöscht sind keine weiteren Aktionen mehr möglich!
